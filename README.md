# Atcoder with Python

## 準備

```bash
./start.sh ABC123
```

を実行すると、ABC123 というディレクトリが生成され、その中に問題ごとの.py ファイルが生成される。

## コードテスト

```bash
python3 sample.py > sample.txt
python3 ABC123/A.py < sample.txt
```

## 実行時間測定

```bash
time python3 ABC123/A.py < sample.txt
```

## git push

```bash
./git_push.sh ABC123
```
